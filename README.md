# Neural Network Toolbox (NN Toolbox)

A comprehensive, full-stack educational and experimental platform for Neural Networks.

## Features

- **Educational Wiki**: In-depth explanations of various NN architectures (Feedforward, CNN, RNN, LSTM, Transformers, Autoencoders, GANs).
- **Hopfield Network**: Interactive pattern memory and recall visualization with resemblance calculation.
- **Sentiment Analysis**: Hybrid training supporting both TensorFlow and PyTorch for text classification.
- **Custom Trainer**: Real-time training monitoring with loss and accuracy graphs.
- **Gradient Diagnostics**: Tools to visualize and debug vanishing/exploding gradients.
- **Attendance System**: OpenCV-based face recognition for automated attendance.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Vanilla CSS.
- **Backend**: FastAPI (Python), TensorFlow, PyTorch, OpenCV.

## Getting Started

### Backend
1. Navigate to `backend/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the server: `python main.py`

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run the dev server: `npm run dev`
